# spring-thymeleaf-demo
Spring Boot Thymeleaf 使用デモ

## 画面一覧

|No.|画面名|パス|
|:--:|--|--|
|1|トップ画面|`/`|
|2|登録画面|`/entry/input`|
|3|登録完了画面|`/entry/complete`|

## 画面遷移図

```mermaid
stateDiagram-v2

[*] --> トップ画面: ユーザーアクセス
トップ画面 --> 登録画面: 登録ページへ<br>リンク押下
登録画面 --> 登録完了画面: 登録する<br>ボタン押下
登録画面 --> トップ画面: トップページへ戻る<br>リンク押下
登録完了画面 --> トップ画面: トップページへ戻る<br>リンク押下
```
